Weyder is a simple weather information application, that is hosted on github, available as a npm module along with cli, as a heroku app, as well as a desktop app.

To view the github repohttps://github.com/Devorein/weyder-webapp

To get the npm module visit https://www.npmjs.com/package/weyder

To use the heroku hosted web app visit https://weyder-devorein.herokuapp.com
